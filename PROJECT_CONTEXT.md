# MyBoss360

## Vision

MyBoss360 is an AI-first Business Operating System for entrepreneurs, executives, consultants and small businesses.

The platform combines:

- CRM
- Tasks
- Calendar
- AI Assistant
- Projects
- Documents
- Finance
- Automations
- Analytics
- Team Collaboration

The design philosophy is inspired by:

- Apple
- Linear
- Notion
- Stripe
- Arc Browser

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Vercel
- Supabase (future)
- OpenAI API (future)

## Design Rules

Minimal.
Modern.
Fast.
Professional.
Large white spaces.
Rounded corners.
Subtle shadows.
Responsive.
Accessibility first.

Never generate unnecessary code.
Always use reusable components.
Never duplicate logic.
Prefer Server Components whenever possible.
Keep folder structure organized.

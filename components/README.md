# Components

Shared React components live here.

- `ui/` contains primitive, reusable interface building blocks.
- `layout/` contains app-wide structural components like headers and shells.
- `marketing/` is reserved for reusable public-site sections and landing page composition.
- `dashboard/` is reserved for authenticated workspace and internal product surfaces.
- `crm/`, `finance/`, `calendar/`, and `ai/` hold domain-specific presentation components.
- `shared/` holds cross-domain UI patterns that do not belong to one module.

Existing working components remain in place until a deliberate migration is scheduled.
